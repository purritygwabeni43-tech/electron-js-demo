# Bestie Chat 💬

A cute desktop chat app built with **Electron.js**, HTML, CSS, and JavaScript.

Bestie Chat is a mini iMessage-style desktop app where users can view contacts, read messages, and send their own chat messages through a simple interface.

## ✨ Features

- Desktop app powered by Electron.js
- iMessage-inspired chat layout
- Sidebar with contacts
- Active chat window
- Message input form
- Custom styling with CSS
- Beginner-friendly project structure

## 🖥️ Preview

The app includes:

- A sidebar titled **Bestie Chat**
- A contact list with names and statuses
- A main chat area for Amahle
- Sent and received message bubbles
- A message form with a “Send the tea” button

## 🛠️ Built With

- CSS
- JavaScript
- Electron.js
- Node.js
